    准备

    建议准备一个干净、换好源的 Ubuntu 16.04 及以上版本（建议 清华源 ），本教程也适用其他 ROS1版本。

    查看ubuntu 版本

    根据自己的 Ubuntu 的版本选择 ROS 版本 (示例是 Ubuntu 18.04 所以对应ROS版本为 melodic）
