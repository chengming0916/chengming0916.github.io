三、安装与卸载

1. 安装

验证效果：

  • 桌面出现快捷方式
  • 开始菜单 → Other 中出现应用
  • 应用文件位于 /opt/Test/

2. 卸载

卸载效果：

  • 删除 /opt/Test/ 目录
  • 自动移除桌面和开始菜单快捷方式
  • （若在 postrm 中配置）删除应用日志

关键注意事项

  1. 依赖完整性：显式加载的 .so 需单独处理依赖
  2. 路径一致性：.desktop 文件中的路径必须匹配安装路径
  3. 脚本权限：DEBIAN 目录下的脚本必须有执行权限
  4. 包名唯一性：control 中的 Package 字段需全局唯一
